About the ePOD Project

The Electronic Proof of Delivery (ePOD) system is an educational project developed as part of the Internet Application and Web Programming course (CPIS-358). It demonstrates how HTML, CSS, and JavaScript can be used to build an interactive, static web application that represents a real-world business process.

Our Mission

The main goal of the ePOD system is to create a simple digital platform for recording proof of delivery events — allowing delivery staff to capture customer signatures, store them securely, and link them to SAP records for audit and reporting purposes.

  • Enhance delivery accuracy and accountability.
  • Reduce paperwork by enabling digital signatures.
  • Integrate delivery confirmation data with enterprise systems.
