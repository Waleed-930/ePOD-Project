Electronic Proof of Delivery (ePOD)

A stand-alone digital platform that replaces paper-based proof-of-delivery with secure electronic signatures, order tracking, and simple performance dashboards.

This is a CPIS-358 educational project that simulates how a real ePOD system could work using HTML, CSS, and JavaScript only.

Go to Login Create an Account

Key Features (Concept)

  • Salesperson or driver can enter or upload sales orders into the system.
  • System generates a digital Proof-of-Delivery (POD) for each order.
  • Customer reviews items and approves using an electronic signature.
  • Signed POD is stored as a secure record with signature, timestamp, and hash.
  • Dashboard shows POD completion by salesperson / customer and pending deliveries.
  • Automated reminders are sent for PODs that are not yet signed.

In this first version, all logic is simulated in the browser with JavaScript only.
